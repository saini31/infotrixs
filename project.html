PROJECT

Medfinder(medicine finder)

Medfinder is a web-based software.The aim of Medfinder is to provide knowledge about medicine availability.

An added feature is the ability to search for a medicine and obtain the address of medical stores.

The user can register their medical store and add the medicine they want to display on web page

Tools/Language

Apache Xampp server,PHP,HTML,CSS,Basic JavaScript,Bootstrap,JQuery,MySql
